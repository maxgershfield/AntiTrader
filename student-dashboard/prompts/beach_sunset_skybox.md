# AntiTrader School — Blockade skybox (beach at sunset)

Equirectangular 360° **beach at sunset** with **distant mountains** — calm background for the student dashboard.

## Prompt (canonical)

The live prompt is **`PROMPT` in `../scripts/generate_skybox.py`** — edit there, then re-run:

```bash
python3 AntiTrader/student-dashboard/scripts/generate_skybox.py
```

## Generate

From repo root. The script loads **`worldgen-oasis-demo/.env`** for `BLOCKADE_LABS_API_KEY` automatically.

This writes **`default-skybox.json`** next to `student-dashboard.html`.
